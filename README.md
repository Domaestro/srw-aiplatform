# Защищённая платформа для разработки и обучения моделей ИИ

Прототип защищённой ML-платформы (MLOps в закрытом контуре): развёртывание,
автоматизация и тестирование. Платформа даёт командам единую защищённую среду
обучения моделей: единый вход (SSO), изоляцию проектов, запуск обучения в кластере,
хранение моделей и артефактов, отслеживание экспериментов и наблюдаемость.

## Стек технологий

| Назначение | Технология |
|---|---|
| Оркестрация контейнеров | Kubernetes (k3d / k3s) |
| Контроль доступа, единый вход | Keycloak (OIDC, SSO), oauth2-proxy |
| Сервисная сеть, TLS | Istio, cert-manager (локальный CA) |
| Ядро машинного обучения | Argo Workflows, Training Operator (PyTorchJob) |
| Отслеживание и реестр моделей | MLflow |
| Хранилища | MinIO (S3), PostgreSQL |
| Мониторинг | Prometheus, Grafana, node-exporter |
| Безопасность Kubernetes | Pod Security Admission, RBAC, NetworkPolicy, аудит |
| Автоматизация и тестирование | Ansible, Molecule, Trivy |

## Структура репозитория

| Каталог | Содержимое |
|---|---|
| `prototype/` | манифесты Kubernetes |
| `ansible/` | роли (по одной на компонент), главный playbook `site.yaml`, инвентарь, сценарий Molecule (`molecule/default`) |
| `tests/` | автоматизированные функциональные и security-проверки (`run_all.sh`, `test_*.sh`) |
